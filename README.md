
## 简介
Element UI表单设计及代码生成器，可将生成的代码直接运行在基于Element的vue项目中。 
- [github pages地址](https://jakhuang.github.io/form-generator)
- [gitee pages地址 较快](https://jakhuang.github.io/form-generator)

## 特点
- 可视化设计表单
- 一键生成原生的vue单页表单代码，100%拓展性
- 在线编辑器修改生成的代码，并实时预览
- 表单栅格化布局
- 支持表单校验
- 快速查阅组件的官方文档
